# Project Vision

## The problem

Every company accumulates knowledge — documents, playbooks, decisions, tribal knowledge — faster than anyone can absorb it. New hires ramp slowly. Experts get interrupted with repeat questions. Institutional memory decays every time someone leaves.

## The vision

**CortexOS turns company knowledge into intelligent AI teams.**

Not a chatbot bolted onto a wiki. An operating system for the AI-native organization:

- **Departments** mirror how the company is actually structured.
- **Knowledge** is centralized, versioned, and searchable.
- **Workflows** capture how work truly gets done.
- **Memory** preserves decisions and context across time.
- **Agents** (Phase 2) operate inside all of the above with real awareness.

## Design principles

1. **Enterprise-first.** Row-level security, org boundaries, and auditability are foundational, not add-ons.
2. **Minimal, premium UI.** Comparable to Linear, Notion, Vercel, Stripe.
3. **Foundations before features.** Ship scaffolding that AI features plug into cleanly, not demos disguised as products.
4. **Boring where it matters.** Predictable auth, predictable data, predictable navigation.

## Non-goals (Phase 1)

- No agent framework yet
- No RAG pipeline yet
- No inference or embedding usage yet
- No billing or plans
